<h2>날씨 앱 (mini project 5)</h2>
유저가 도시 이름을 입력하면 weather api를 받아와서 날씨를 표시하는 앱
<br> <br>
<div>
유저가 정확한 도시 입력시 : 해당 도시의 날씨 출력<br>
- icon : 날씨 아이콘 <br>
- temperature : 현재 온도 <br>
- min & max temperature : 최저기온 / 최고기온 <br>
- humidity : 습도 <br>
- wind speed : 풍속 <br>
</div>
 <br>
<div>
유저가 틀린 도시 이름 입력시 : city not found 출력
</div>
